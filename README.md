# Car-Rental

# 🚗 Car Rental System – MERN Stack

A full-stack **Car Rental Web Application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
The platform allows users to browse cars, book rentals, and manage bookings, while owners/admins can manage cars and reservations.

---

## 📌 Features

### 👤 User Features
- User authentication (Register / Login)
- Browse available cars
- View car details
- Book cars for specific dates
- View personal bookings
- Password reset with OTP verification

### 🧑‍💼 Owner / Admin Features
- Owner login
- Add new cars with images
- Manage car listings
- Manage bookings
- Dashboard with booking overview

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Context API
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (file uploads)

### Database
- MongoDB (Mongoose ODM)
